# codexU v1.0.1

首个 1.0 正式版，聚焦兼容性与额度可视化。

- 兼容新版 ChatGPT/Codex App 的动态路径，同时保留旧版 App 与标准 CLI 回退。
- 双环额度新增低开销逆时针粒子流，只在剩余额度弧段内运动，并支持“减少动态效果”。
- 关闭主窗口后台运行时隐藏 Dock 图标，从菜单栏或快捷键唤回时恢复。

## 安装包

- Apple Silicon：`codexU-1.0.1-mac-arm64.dmg`
- Intel：`codexU-1.0.1-mac-x86_64.dmg`
